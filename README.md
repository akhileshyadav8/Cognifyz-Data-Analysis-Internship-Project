# Cognifyz Data Analysis Internship Project

## 📌 Internship Overview
This project is part of the **Data Analysis Internship Program by Cognifyz Technologies**.  
The objective of this internship is to apply data analysis concepts on a real-world restaurant dataset and extract meaningful insights using Python.

The project covers **data cleaning, exploratory data analysis, visualization, and insight generation** across multiple analytical levels.

---

## 🛠️ Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset Information
- Dataset contains restaurant-level data including:
  - City
  - Cuisines
  - Ratings
  - Votes
  - Price Range
  - Online Delivery
  - Table Booking
  - Latitude & Longitude
- The dataset was cleaned before analysis to handle missing values, incorrect formats, and inconsistent entries.

---

## 🧹 Data Cleaning Steps
- Removed rows with missing critical values (city, cuisines, ratings)
- Filled missing numerical values using logical defaults
- Standardized column names
- Processed multi-valued cuisine fields for analysis

---

## 📊 Tasks Performed

### 🔹 Level 1
- Identified top 3 most common cuisines and their percentages
- Analyzed city-wise restaurant distribution and ratings
- Visualized price range distribution
- Compared ratings based on online delivery availability

---

### 🔹 Level 2
- Analyzed rating distribution and average votes
- Identified common cuisine combinations and their ratings
- Performed geographic analysis using latitude and longitude
- Identified restaurant chains and analyzed their popularity

---

### 🔹 Level 3
- Analyzed review length and its relationship with ratings
- Studied vote distribution and correlation with ratings
- Compared price range with online delivery and table booking availability

---

## 📈 Key Insights
- Certain cuisines dominate the restaurant market
- Cities with fewer restaurants often have higher average ratings
- Higher price range restaurants are more likely to offer online delivery and table booking
- Votes and ratings show a positive correlation
